# Lox programming language
This is a lox programming language implementation written in C.

# Run the interpreter
**clone the repo*
```git clone https://github.com/VladPetryshyn/clox.git && cd clox/src```
**build the interpreter**
```make build```

**run the repl**  
```./clox.o```
**execute the file**  
```./clox.o test.lox```
